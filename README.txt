To execute Part I of this project:

1) Extract the zipped folder
2) Be in the top-level directory (if you use 'ls' from this directory you should see tons of files)
3) Enter the command: 'python3 part1.py'. Please be sure to enter 'python3' and not 'python' as this code won't run on Python 2.7.

1) Be in the top-level directory (if you use 'ls' from this directory you should see tons of files)
